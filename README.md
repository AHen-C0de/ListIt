App to create custom shopping and TODO lists.

# Projekt SetUp

## VSCode
- set python interpreter:
    1) set `"python.defaultInterpreterPath"` in `setting.json` (with *pylance* extension) to the python executable in your venv `/venv/bin/python`
    2) If it doesn't work, also set it manually via the GUI > command line > "Python: Select Interpreter" > `<path-to-python>`
